# AadhaarInsight360

Comprehensive data analytics platform built for the UIDAI Data Hackathon 2026, providing real-time insights into Aadhaar enrolment and update activities across India.

## Features

- Geospatial analysis of enrolment patterns across states and districts
- Time-series trend analysis of Aadhaar registrations
- Demographic segmentation and cohort analysis
- Interactive dashboards with drill-down capability
- Policy impact simulation
- Automated report generation

## Tech Stack

Python, Streamlit, Plotly, Pandas, GeoPandas, NumPy

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/AadhaarInsight360.git
cd AadhaarInsight360
pip install -r requirements.txt
```

### Usage

```bash
streamlit run app.py
```
